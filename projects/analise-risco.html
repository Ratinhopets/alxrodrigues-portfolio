Visualização Interna (Chart.js)

Chart.js Análise de Risco 2024

Este projeto foi desenvolvido para demonstrar a capacidade de carregar visualizações complexas dentro da própria interface do portfólio, sem a necessidade de recarregar a página.

O Desafio

Muitas vezes, relatórios atuariais e de risco precisam ser apresentados de forma concisa, mas com a opção de aprofundamento (drill-down). A estrutura aqui utilizada simula um dashboard executivo.

Metodologia

  • Coleta de dados via API proprietária.
  • Tratamento de outliers utilizando Python (Pandas).
  • Visualização dinâmica renderizada no cliente.

Ficha Técnica

  • Linguagem: JavaScript / Python
  • Tipo: Web App
